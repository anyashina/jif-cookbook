## Budget Framework in Section 3: Key Takeaways for Next Director

1. **Per-Fellow Costs:** Start with $20,300 (Jerusalem model), adjust for your city's cost of living. Use Kyiv/Krakow/Chicago as reference brackets.

2. **Staffing is Non-Negotiable:** You need 5 core roles (ED, PD, Communications, Finance, Coordinators). Compressed hiring (2022 style) results in 30-68% understaffing.

3. **Optimal Scale:** 6 Fellows is more efficient than 4 (per-Fellow cost drops 15% while adding minimal fixed costs).

4. **Contingency Required:** 10% budget reserve ($15K+ for $156K program) for mid-program adjustments.

5. **Funding Diversification:** No single funder should be >50%. Russell Berrie + Bloomberg are anchors; also pursue individual donors, mid-level foundations, in-kind support.

6. **Critical Path:** Major fundraising begins 18-24 months before program launch. Program director hiring 12-16 weeks before. Communications manager 8-12 weeks before.
