# JIF Cookbook Primary Sources

This directory contains all primary source materials for the Jerusalem International Fellows Cookbook — evaluations, transcripts, MailChimp campaigns, press coverage, and video documentation.

## Folder Structure

- **00-evaluations-reports/** — Independent evaluation by On Track Consulting, program assessments
- **01-timeline/** — MailChimp campaign emails, press releases, historical documents
- **02-kitchen-notes/** — Interview transcripts, stakeholder recordings, Ruth Cummings materials
- **03-recipes/** — Operational documentation, planning materials, organizational records
- **04-network/** — People directory, relationship maps, host organization materials
- **05-vision/** — Elise's reflections, strategic documents, founding materials

## File Organization

Files are organized by source type and section relevance. Large video files (>100MB) are stored using Git LFS.

### Files Already Included

- **2022-on-track-evaluation.pdf** — 52-page independent evaluation by On Track Consulting (June 2022)
- **ruth-cummings-transcript.txt** — Zoom recording transcript with Ruth Cummings
- MailChimp campaign archives (key communications 2021–2026)
- Video documentation from 2022 residency
- Press coverage and media archives

## How to Add Files

Files are added via GitHub's web interface or through Git. Large files (video, audio) will be tracked via Git LFS automatically through the `.gitattributes` configuration.

## Transparency & Citation

These sources support every claim in the Cookbook. Citation links point directly to these files on GitHub, ensuring readers can verify sources and understand how the program evolved.
