# For Replicators

This folder contains operational manuals and frameworks for launching the program in a new city.

- **Methodology** - The five guiding questions and core philosophy
- **Budgets** - Scalable cost templates for three economic tiers
- **Staffing** - Organizational structure and recruitment timelines
- **Host Organization Vetting** - Partnership evaluation rubric
- **Selection Process** - Fellow recruitment and vetting lifecycle
